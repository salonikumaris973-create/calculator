# Calculator

## Description

A simple and responsive calculator built using HTML, CSS, and JavaScript. This calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an easy-to-use interface.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear button (C)
* Responsive design
* User-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

calculator/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `index.html` or open it in your web browser.

## Usage

Enter numbers using the calculator buttons, choose an arithmetic operator, and press "=" to view the result.
